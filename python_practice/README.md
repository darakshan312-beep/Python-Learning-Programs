# Python Fundamentals Practice

This repository contains my learning journey and practice work for the fundamentals of the Python programming language. The purpose of this repository is to build a strong foundation in Python through hands-on coding exercises and practical examples.

## Topics Covered

### 1. Variables

* Creating and using variables
* Naming conventions
* Assigning values

### 2. Data Types

* Integer (`int`)
* Float (`float`)
* String (`str`)
* Boolean (`bool`)

### 3. Operators

* Arithmetic Operators
* Comparison Operators
* Logical Operators
* Assignment Operators

### 4. Conditional Statements

* `if` statement
* `if-else` statement
* `if-elif-else` statement
* Nested conditions

### 5. Loops

* `for` loop
* `while` loop
* Loop control statements:

  * `break`
  * `continue`
